## Запуск проекта
1. Клонируйте репозиторий.
2. Скорректируйте параметры в файле конфигурации SongLibrary\configs\songLibrary.env для подключения для БД и внешнего API сервера.
3. Создайте на сервере БД новую БД с именем 'song_library'.
4. Запустите проект.
